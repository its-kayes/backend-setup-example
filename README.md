1. Clone Project. 
2. For Install dependencies and Packages. 

    ``` bash  
    yarn add 
3. For Run Project. 

    ``` bash  
    yarn start
4. For Build Project. 

    ``` bash
    yarn build
5. For Test Project. 

    ``` bash
    yarn test
6. For Lint Project. 

    ``` bash
    yarn lint
7. For Format Project. 

    ``` bash
    yarn format
8. For Analyze Project. 

    ``` bash
    yarn analyze
10. For Deploy Project. 

    ``` bash
    yarn deploy -- --dry-run
